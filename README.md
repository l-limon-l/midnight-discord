# midnight

a dark, customizable discord theme.

## install

### vencord/betterdiscord (or any client that supports theme files)

1. download the theme file, [`midnight.theme.css`](https://github.com/l-limon-l/midnight-discord/blob/main/themes/midnight.theme.css). (there should be a download button at the top right of the page)
2. drag the file into your theme folder. (there should be a button to open the theme folder in theme settings)
3. (optional) customize the theme by editing the options in `midnight.theme.css`.

### install through link

add `https://l-limon-l.github.io/midnight-discord/build/midnight.css` to your theme import links. you will need to copy the theme variables to your quickcss in order to customize the theme.
